# agri_forecast_seasonal
# 순 데이터 배추 가격 예측
# version 1,2,3

## 배추 가격 예측 모델 (순 단위)

순 단위(10일 간격)로 전처리된 기상 데이터와 가격 데이터를 사용하여 배추 가격(7일/1순 후)을 예측합니다.

### 모델 버전 요약

| 버전 | 설명 | 기상 변수 특징 |
| :--- | :--- | :--- |
| **Version 1** | **변수별 최적화** | 상관분석을 통해 각 변수별로 가장 높은 상관관계를 보인 **누적 기간(N순)**과 **시차(Lag)**를 개별 적용했습니다.<br>(예: 평균기온 24순/Lag9, 강수량 9순/Lag0 등) |
| **Version 2** | **고정 (Short-term)** | 모든 기상 변수에 대해 동일하게 **누적 3순 (약 1개월)**, **시차 3순**을 적용했습니다. |
| **Version 3** | **고정 (Mid-term, No Lag)** | 모든 기상 변수에 대해 동일하게 **누적 6순 (약 2개월)**, **시차 0순**을 적용했습니다. |

<br>

### 성능 비교 결과 (2025년 테스트셋 기준)

| 모델 | MAE (오차) | RMSE | R² (결정계수) | MAPE (오차율) |
| :--- | :--- | :--- | :--- | :--- |
| **Version 1** | **1,891원** | 2,498원 | 0.4220 | 18.65% |
| **Version 2** | 2,010원 | 2,653원 | 0.3450 | 19.23% |
| **Version 3** | 1,906원 | 2,492원 | 0.4248 | 18.57% |

* **결과 분석**: 각 변수별로 최적의 누적 기간과 시차를 찾아 적용한 **Version 1**이 전반적으로 가장 안정적인 성능(MAE 기준)을 보였습니다. 단순 고정 기간을 사용한 Version 3(누적 6순)도 비교적 양호한 성능을 보였으나, Version 2(누적 3순)는 다소 성능이 떨어졌습니다.
* **사용 모델**: XGBoost (Version 1, 2, 3 모두 XGBoost가 LSTM보다 우수한 성능을 보여 최종 선택됨)
